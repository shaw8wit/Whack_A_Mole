## Getting Started

Clone the repo or download as zip.
Start ```main.html``` on any server (e.g. localhost).

## Built with

Vanilla JavaScript, Html, and CSS.

## Features

+ Set Difficulty
+ Set Duration
+ Smooth gameplay with animations
+ Saves highscore locally
